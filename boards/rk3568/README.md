# RK3568 Board Track

RK3568 is the planned hardware track for later labs. Keep board-specific code here and avoid mixing it into the QEMU path.

Expected future content:

- U-Boot and kernel build notes;
- device tree overlays and board DTS patches;
- flashing and serial recovery guide;
- GPIO, I2C, UART, PWM validation scripts;
- platform adapter for `edge-agent`.

The first course milestone still uses QEMU so every student and CI job has the same baseline.
